| Connective      | Symbol            | Formal name |     
| --------------- | ----------------- | ----------- | 
| and             | $\land$           | conjunction |     
| or              | $\lor$            | disjunction |     
| not             | $\neg, \bar{A}$   | negation    |     
| if ... then ... | $\Rightarrow$     | implication |     
| if, and only if | $\Leftrightarrow$ | equivalence            |
